# 启动 @run

```bash
npm run init

npm run dev # 启动前端

npm run start # 启动客户端
npm run start:cef # 或者兼容模式启动客户端
```

# 打包 @build

```bash
npm run build
npm run build:cef # 或者兼容模式
```
